# 桃桃拟人化与 3D 动态生成规格

## 核心设定

桃桃是情侣共同唤醒的小伙伴。它可以来自宠物、纪念物、玩偶、礼物、共同照片里的小物件，也可以来自系统默认形象。唤醒后的桃桃不是一次性头像，而是会动、会说话、会参与日常决定和共同回忆的动态小伙伴。

界面可见文案使用：

- `唤醒桃桃`
- `让这张照片变成你们的小伙伴`
- `给桃桃一点性格`
- `看看桃桃动起来的样子`

## 照片唤醒目标

上传照片不是为了做头像，而是为了建立情感来源。

好的上传对象：

- 两个人都喜欢的宠物。
- 第一次约会买的东西。
- 共同养的植物。
- 玩偶。
- 礼物。
- 有共同意义的物件。
- 合照中的某个非敏感对象。

不推荐：

- 纯自拍。
- 过度私密照片。
- 证件照。
- 容易引发隐私风险的照片。

## 唤醒流程

### Step 1：选择照片

页面文案：

`上传一张你们都有感觉的照片，桃桃会从这里醒来。`

支持：

- 拍照。
- 相册选择。
- 默认形象。

可选来源：

- 宠物照片。
- 纪念物照片。
- 玩偶或礼物照片。
- 共同照片里的小物件。
- 共同相册里已经保存过的照片。

从共同相册带回唤醒时，要保留照片来源和已识别对象，不让用户重新从空白开始。

### Step 2：识别情感对象

系统识别照片中的主要对象，并让用户确认：

- 宠物。
- 物件。
- 植物。
- 玩偶。
- 礼物。
- 其他。

页面文案：

`桃桃想从这个小东西里醒来，可以吗？`

### Step 3：选择拟人化方向

选项：

- 可爱。
- 温柔。
- 活泼。
- 黏人。
- 安静。
- 小太阳。

这里不是选择 AI 模型，而是选择桃桃的脾气。

### Step 4：生成动态桃桃

首发可以使用 2.5D/3D 风格动态角色，不要求一开始真实可编辑 3D 模型。

首发可接受形态：

- 3D 风格静态形象。
- 轻动效表情。
- 闲置动作。
- 眨眼。
- 开心挥手。
- 投喂反馈。

后续升级：

- 图像转 3D 角色。
- 角色骨骼绑定。
- 多表情。
- 多动作。
- 小窝中实时互动。

### Step 5：设置名字与口吻

默认名为 `桃桃`，允许用户改名。

命名规则：

- 名字不能为空；未填写时使用 `桃桃`。
- 一旦保存，首页、三方群聊、小窝、投喂弹层、我们页、小程序、旅程、提醒、相册、记小事和回忆详情都使用同一个名字。
- 二次唤醒或在我们页调整时，应保留当前名字，除非用户主动修改。
- 不同页面不能各自写死 `桃桃`，必须读取同一份桃桃资料。

说话风格：

- 甜一点。
- 皮一点。
- 温柔一点。
- 像小孩一点。
- 像宠物一点。
- 少说教。

避免选项：

- 专业权威式角色。
- 诊断关系式角色。
- 医疗咨询式角色。
- 判断输赢式角色。

### Step 6：双方确认

另一半需要确认：

- 桃桃形象。
- 桃桃名字。
- 桃桃说话风格。

确认文案：

`桃桃以后就是你们共同养的小伙伴啦。`

确认后形成一份全局资料：

- 名字。
- 来源照片。
- 醒来的对象。
- 性格方向。
- 说话方式。
- 动作。
- 形象。
- 是否已由另一半确认。

这份资料要同步到 App、小程序和提醒回流，不因为从微信进入、重新登录或回到相册而丢失。

## 桃桃状态

首页可展示三个轻量状态：

- 心情。
- 活力。
- 小愿望。

示例：

- 心情：很开心。
- 活力：88%。
- 小愿望：想和你们贴贴。

状态不能变成惩罚或评分。

禁止：

- 你们今天让桃桃失望了。
- 亲密值下降。
- 对方没有完成任务。
- 桃桃不开心是因为你。

## 桃桃动作

首发动作：

- 开心。
- 眨眼。
- 挥手。
- 期待。
- 被投喂。
- 收到礼物。
- 记录回忆。
- 陪你们想晚饭。

投喂反馈：

- 桃桃开心地转一圈。
- 小窝出现新物件。
- 桃桃写一张感谢小纸条。
- 桃桃说一句短短的撒娇话。

## 桃桃文案原则

桃桃要像一个小生命说话，而不是像系统提示。

推荐：

- `桃桃今天很开心，想和你们一起决定晚饭。`
- `这句话有点硬，桃桃帮你变软一点。`
- `桃桃记住啦，这是你们的小惊喜。`
- `你们都有好吃的，桃桃也想要一颗小布丁。`
- `不给也没关系，桃桃还是会陪你们。`

禁止：

- `AI 已为你生成建议。`
- `正在分析你们的关系状态。`
- `你们的心灵信号正在同步。`
- `根据关系模型判断。`
- `对方更应该道歉。`

## 小窝与成长

小窝是桃桃生活的地方，不是商城。

小窝可承载：

- 背景。
- 小床。
- 灯。
- 零食。
- 节日装饰。
- 礼物。
- 回忆摆件。

成长不使用等级压迫。

可用表达：

- `桃桃的小窝变暖了一点。`
- `桃桃多记住了一个你们的小习惯。`
- `桃桃今天学会了你们喜欢吃辣。`

避免：

- 等级排行榜。
- 亲密分惩罚。
- 未投喂就变差。
- 逼迫式成长。
